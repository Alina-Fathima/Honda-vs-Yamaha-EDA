EDA and Machine Learning on Honda vs Yamaha bike dataset using Python, Pandas, Seaborn and Scikit-learn
# Honda vs Yamaha Bike EDA & ML Analysis

## Overview
Comparative Exploratory Data Analysis on Honda and Yamaha 
motorcycles to identify performance patterns and predict 
bike ratings using Machine Learning.

## Dataset
- 100 records (50 Honda + 50 Yamaha)
- Features: Engine CC, Mileage, Price, Top Speed, Weight, 
  Launch Year, Rating

## Tools & Libraries
Python | Pandas | NumPy | Matplotlib | Seaborn | Scikit-learn

## Steps Performed
1. Data Loading and Merging
2. Data Cleaning (null values, duplicates)
3. EDA with 5 visualization types
4. Correlation Heatmap
5. ML Models: Linear Regression, Decision Tree, Random Forest

## Key Findings
- Honda bikes are slightly higher priced on average
- Mileage is similar across both brands (~53 kmpl)
- Price is the most important feature for rating prediction
- Most bikes priced between 1-3.5 lakhs

## Visualizations
- Bar Chart: Average price and mileage comparison
- Histogram: Price distribution
- Boxplot: Price spread by brand
- Scatter Plot: Mileage vs Price
- Correlation Heatmap
- Feature Importance Charts
